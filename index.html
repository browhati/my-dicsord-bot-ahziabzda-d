About Zen Bot

Zen Bot is a feature-packed bot designed to enhance your Discord server's functionality. From automating moderation to providing fun commands, Zen Bot is here to make your server better!

Features

  • ?echo - Repeat any message in the channel
  • ?stats - View the bot's usage statistics
  • ?prefixes - Manage custom prefixes
  • ?coinflip - Flip a coin
  • ?reactionroles - Assign roles using reactions
  • ?kick, ?ban, ?mute - Manage server moderation
